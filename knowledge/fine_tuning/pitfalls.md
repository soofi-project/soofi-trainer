# Fine-Tuning Pitfalls

## Overfitting

Ein häufiger Fehler beim Fine-Tuning ist Overfitting auf kleinen Datensätzen. Das Modell lernt die Trainingsdaten auswendig, statt zu generalisieren.

## Catastrophic Forgetting

Zu aggressives Fine-Tuning kann dazu führen, dass das Modell allgemeines Wissen verliert. Niedrige Lernraten und parametereffiziente Methoden wie LoRA können dies abmildern.
