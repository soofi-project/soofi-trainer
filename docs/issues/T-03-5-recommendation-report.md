# Description

**Structured Recommendation Report**

The agent generates a structured output with primary recommendation, alternatives, and next steps.

## Report Structure

### 1. Primary Recommendation

```markdown
## 🎯 Primary Recommendation: [Method]

### Reasoning
[Why this method is optimal for the use case]

### Estimated Resources
- **Hardware**: [GPU type, VRAM requirements]
- **Training Time**: [Estimated duration]
- **Cost**: [Approximate cost if cloud]

### Recommended Dataset
[If HuggingFace search was performed]
- Name: [Dataset Name]
- Size: [Number of samples]
- Link: [HuggingFace URL]
```

### 2. Alternative(s)

```markdown
## 🔄 Alternative(s)

### Alternative 1: [Method]
**When suitable**: If [condition changes]
**Trade-offs**: [Pros/cons compared to primary recommendation]

### Alternative 2: [Method]
**When suitable**: If [other condition]
**Trade-offs**: [Pros/cons]
```

### 3. Next Steps

```markdown
## ✅ Next Steps

1. **Prepare Data**
   - [Specific instructions for data format]
   - [Quality requirements]

2. **Set Up Infrastructure**
   - [Hardware recommendations]
   - [Software/libraries]

3. **Start Training**
   - [Recommended hyperparameters]
   - [Monitoring tips]

4. **Evaluation**
   - [Relevant metrics]
   - [Benchmark recommendations]
```

## Report Generation

```python
class ReportGenerator:
    def generate_report(self,
                       user_context: UserContext,
                       recommendations: List[Recommendation],
                       datasets: Optional[List[Dataset]] = None) -> str:
        """Generate the complete recommendation report."""

        primary = recommendations[0]
        alternatives = recommendations[1:3]

        report = RECOMMENDATION_REPORT_TEMPLATE.format(
            summary=self._generate_summary(user_context, primary),
            primary_method=primary.method,
            primary_reasoning=primary.reasoning,
            primary_score=primary.score,
            score_breakdown=self._format_scores(primary.criteria_scores),
            hardware_requirements=primary.hardware,
            estimated_time=primary.time_estimate,
            estimated_cost=primary.cost_estimate,
            dataset_recommendation=self._format_dataset(datasets),
            alternatives=self._format_alternatives(alternatives),
            next_steps=self._generate_next_steps(primary, user_context),
            timestamp=datetime.now().isoformat()
        )

        return report
```

## Output Formats

The report can be output in the following formats:
- **Markdown** (default for chat)
- **JSON** (for programmatic processing)
- **Plain Text** (for simple display)
